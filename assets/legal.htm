Personal Data Privacy

Since Kids Fables does not require registration there is no need to collect and process personal private data. The only exception are suggestion, developer message and rights violation forms that request for sender email address. Anyway, email address is optional and needed only if sender wishes for reply.

Kids Fables guarantees that email address, if provided, is used only to reply and never to send unsolicited emails. No third party spamer service will benefit from collected email addresses.

Cookies

The Cookie Law was designed to protect online privacy, by making consumers aware of how information about them is collected and used online, and give them a choice to allow it or not.

Kids Fables uses cookies to gather anonymous data about application usage. No user data is collected or processed, only traffic data that help improving user experience. Anyway, user still has the option to disable cookies, for Kids Fables site only or globally, using browser specific settings. Kids Fables will continue working with no degradation.
